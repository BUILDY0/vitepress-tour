# VitePress 介绍

## 什么是 VitePress？

VitePress 是一个基于 **Vite** 和 **Vue** 的静态站点生成器（SSG），专为编写技术文档而设计。它是 Vue.js 生态中 VuePress 的继任者，由 Vue 和 Vite 的作者尤雨溪主导开发。

## 核心技术栈

| 技术 | 说明 |
|------|------|
| **Vite** | 新一代前端构建工具，提供极速的开发服务器和构建能力 |
| **Vue 3** | 渐进式 JavaScript 框架，支持 Composition API 和单文件组件 |
| **Markdown** | 文档编写语言，VitePress 对其做了大量扩展 |
| **Vue Router** | 支持基于文件的路由，自动生成侧边栏和导航 |

## VitePress vs VuePress

| 特性 | VitePress | VuePress |
|------|-----------|----------|
| 构建工具 | Vite | Webpack |
| 开发热更新 | 毫秒级 | 秒级 |
| Vue 版本 | Vue 3 | Vue 2/3 |
| 配置方式 | `config.mts` | `config.js` |
| 默认主题 | 更现代、更简洁 | 传统风格 |
| Bundle 大小 | 更小 | 较大 |

## 适用场景

- **项目文档**：开源项目的使用文档、API 文档
- **技术博客**：个人或团队技术博客
- **知识库**：团队内部知识管理和 Wiki
- **产品手册**：软件产品的帮助中心和使用指南

## 为什么选择 VitePress？

1. **极速体验**：基于 Vite 的即时热更新，开发体验极佳
2. **Vue 原生支持**：在 Markdown 中直接使用 Vue 组件，实现交互式文档
3. **开箱即用**：内置搜索、国际化、暗色模式等常用功能
4. **主题灵活**：可轻松自定义主题，也可使用默认主题
5. **SEO 友好**：静态 HTML 预渲染，对搜索引擎友好
6. **活跃社区**：作为 Vue 官方推荐方案，社区活跃，持续迭代
